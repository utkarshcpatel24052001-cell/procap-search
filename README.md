## Installation
To install the BACPOP project, follow these steps:
1. Clone the repository: `git clone https://github.com/utkarshcpatel24052001-cell/procap-search.git`
2. Navigate to the project directory: `cd procap-search`
3. Install required dependencies: `npm install`

## Usage
To run the application, use the following command:
```bash
npm start
```
